Cyjb
====

我开发的 C# 基础类库，以 BSD 协议发布。基于 .Net Framework 4.5，使用 Visual Studio 2013 作为开发工具。

本项目包含一些常用的基础类，主要功能包括：

- 常用功能：
  - 对数字、字符串和枚举等的扩展方法。
  - 常用的基本类型，如文本-值对、三态枚举等。
  - 常用的异常辅助方法。
  - 更易于使用的泛型集合基类。
  - 对集合类的扩展。
  - 对输入输出类的扩展。
- 对反射的高级支持：
  - 更强大的类型成员查找。
  - 简单的动态创建委托。

详细的类库文档，请参见 [Wiki](https://github.com/CYJB/Cyjb/wiki)。
